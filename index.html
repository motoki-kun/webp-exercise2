    HTML要素の説明

  • span要素

    • CSSで装飾をしたいときによく利用する「意味を持たない要素」のことです．

    ここでいう装飾とは文字を大きくしたり，色を変えたりなどといった見た目に関する デザインを行うことです．

    意味を持たない要素にはdiv要素が存在するがspan要素との違いを説明すると div要素はブロック要素であり，span要素はインライン要素であるため，div要素は幅と 高さが指定できるのに対してspan要素はそれができません．また，div要素は要素の終わりに 必ず改行がされるのですが，span要素は改行が行われません．

  • figure要素

    • figure要素は画像やプログラミングコードにキャプションをつけたい際に使う 要素のことである

    figure要素で画像と一緒にキャプションを囲うことによって，画像の位置を変えたいときに キャプションごと位置を変えられるというメリットがあります．

  • table要素

    • table要素は表やカレンダーを作成する際に使われる要素です．

    そして，tableを使う際には基本的にtr要素，th要素，td要素を用いて表を作成していきます． tr要素は表の行を表しており，th要素は表の見出しセルを表し，td要素は表の内容セルを表しています

    マークアップ例

    <h3>たろちゃんの成長</h3> <figure> <img src = "images/taro.png" alt = "たろちゃんの写真" width = "100" height = "150"> <figcaption>たろちゃんの写真</figcaption> </figure> <table border="1"> <tr> <th>年齢</th> <th>体重</th> </tr> <tr> <td>3ヶ月</td> <td>7.9kg</td> </tr> <tr> <td>6ヶ月</td> <td>21.8kg</td> </tr> <tr> <td><span style = "color:red">9ヶ月</span></td> <td><span style = "color:red">2kg</span></td> </tr> </table>
